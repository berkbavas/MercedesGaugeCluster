# Mercedes Gauge Cluster
Written with `QML`.
Indicators were designed by my sister [Gamze Bavaş](https://www.linkedin.com/in/gamze-bava%C5%9F-8100131a1/).

## Build
1) Install `Qt 5.15.2`.
2) Open `MercedesGaugeCluster.pro` with `QtCreator` and build & run it.
